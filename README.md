砂糖・塩などの調味料の重さ測定し可視化する。

今まで不明瞭であった、
今の一振りは実際どれだけの調味料だったのか？
という部分を可視化・データ化する。

さらに、
LINE API・GASを利用することで
LINEボット、angular WEBアプリケーションとして
健康管理を促す。


## 注意点

本コードはこのままcloneやpullしても動きません。
（本番時のコードは別で管理中です。詳細は管理者まで）
